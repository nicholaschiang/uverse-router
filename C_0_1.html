Device access code

Note: "Hide/Show Access Code: These buttons allow you to hide or display the device access code on this and other pages. This setting also controls the display or hiding of Wi-Fi passwords."

Access codes must be at least 8 characters in length, and must consist of characters from at least two of the following sets: alphabetic characters, decimal digits, and the special characters: @ # % & * = ? / < and >.

Custom access code setting
Note: When you choose to protect the device settings with a custom access code, you must also set up an access code hint in case you forget your access code. This hint can be a word, phrase, or question that will help you remember your access code if you forget it. This hint should not contain the access code itself.
