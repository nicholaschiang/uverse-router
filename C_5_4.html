IP Utilities & Tests
Note: Entering the "Host Address" requires either a hostname (e.g. 'www.example.com') or an IP address. The IP address format can be IPV4 (e.g. 'xxx.xxx.xxx.xxx', where xxx represents the number (0-255)) or IPV6 (e.g. 'xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx', where each x represents a 4 bit hex value (0-9, A-F)).
Times or Hops
The default value of 30 will take around 5 minutes if the continuity test fails. To reduce the amount of time, decrease the test depth. Each test depth takes around 10 seconds to timeout.
