System & Link Resets

Clears the device list.
Devices will be re-listed as the system rediscovers them. Clearing the device list deletes many per-device settings you may have made (IP addresses, host application mappings, etc). It is recommended you only do this when instructed to do so by customer care.
Refreshes the Broadband IP Address
Reestablishes your broadband link (physical and logical)
Reboots your system

Configuration Resets

Resets the Wi-Fi AP settings on the gateway to factory defaults
Resets the firewall configuration to factory defaults

Reset to Factory Default State

Warning!
Resetting the system to the factory default state may cause your broadband connection to not function properly and may require reconfiguration of the system to reconnect the broadband service.
Erase all saved changes.
This will revert all configuration parameters to their default state.
